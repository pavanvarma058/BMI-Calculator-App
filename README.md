# BMI Calculator App

A simple and user-friendly **BMI Calculator Android App** built using **Java** and **XML** in **Android Studio**.  
This application allows users to enter their **height** and **weight** and instantly calculate their **Body Mass Index (BMI)** along with the corresponding health category.

## Features

- Clean and simple UI
- Input height and weight
- Calculates BMI instantly
- Displays BMI result
- Shows BMI category:
  - Underweight
  - Normal weight
  - Overweight
  - Obese
- User-friendly Android app built with Java and XML

## Tech Stack

- **Android Studio**
- **Java**
- **XML**

## How It Works

1. User enters height and weight
2. App calculates BMI using the formula:

   `BMI = weight (kg) / height² (m²)`

3. The result is displayed on the screen
4. The app also shows the BMI category based on the calculated value

## BMI Categories

- **Below 18.5** → Underweight
- **18.5 - 24.9** → Normal weight
- **25.0 - 29.9** → Overweight
- **30.0 and above** → Obese

## Project Structure

- **Java** → Handles the BMI calculation logic
- **XML** → Designs the user interface
